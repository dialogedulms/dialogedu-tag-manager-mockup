# dialogEDU Tag Manager Mockup

Interactive mockup for the Tag Manager feature.

## Features
- View tags with usage counts
- Filter by context and usage level
- Search tags by name
- Sort by any column
- Rename, merge, and delete tags (with confirmation modals)

## Deploy to Vercel

This project is ready to deploy. Simply push to GitHub and connect to Vercel.

## Local Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
